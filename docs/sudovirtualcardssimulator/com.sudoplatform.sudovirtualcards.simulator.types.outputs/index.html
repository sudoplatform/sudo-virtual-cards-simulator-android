CurrencyAmount

 data class CurrencyAmount : Parcelable

A currency available in the virtual card simulator. The conversion rates are expressed as Int that are the relative values between all the currencies. To convert one currency to another find the conversion rates of the two currencies, convert to Double, divide them then use that relative value to perform the currency conversion.

SudoVirtualCardsSimulatorClient

interface SudoVirtualCardsSimulatorClient

Interface encapsulating a library for interacting with the Sudo Platform Virtual Cards Simulator service.

This interface allows you to simulate the use of a virtual card at merchants to generate transaction events. The simulator is only available in sandbox environments.

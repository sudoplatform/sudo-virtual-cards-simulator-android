setAppSyncClient

fun setAppSyncClient(client: AWSAppSyncClient): SudoVirtualCardsSimulatorClient.Builder

Provide an AWSAppSyncClient for the SudoVirtualCardsSimulatorClient to use (optional input). If you do not supply this value an AWSAppSyncClient will be constructed and used. If you provide the AWSAppSyncClient via this method you do not need to provide the context, username and password.

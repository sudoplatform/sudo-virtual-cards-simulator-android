 fun build(): SudoVirtualCardsSimulatorClient

Construct the SudoVirtualCardsSimulatorClient. Will throw a NullPointerException if the context, username or password are needed and have not been provided. Will also throw NullPointerException if the configuration file does not supply the adminApiService section with the elements apiUrl, poolId, region and clientId.
